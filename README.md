# Mouse Jiggler

Simple Python script that prevents your PC from going idle by moving the mouse and clicking periodically.

## Features
- Moves mouse and clicks
- Returns cursor to original position
- Lightweight and easy to use

## Requirements
- Python 3
- pyautogui (`pip install pyautogui`)

## Usage

1. Clone or download the repository.
2. Install the dependency:
   ```bash
   pip install pyautogui

Run the script:bash

python mouse_jiggler.py

How to StopPress Ctrl + C in the terminal, or move your mouse to the top-left corner of the screen.
